# **App Name**: CircuitView

## Core Features:

- User Authentication: Secure user sign-up and login functionality to protect electrical data.
- CSV Data Upload: Enable users to upload CSV files containing electrical system parameters.
- Automated Data Extraction: Read CSV headers to automatically detect and list available parameters.
- Dynamic Time-Based Graphs: Visualize selected parameters over time with interactive graphs (Daily, Weekly, Monthly, Yearly).
- Indian Standard Time Conversion: Automatically convert all data timestamps to Indian Standard Time (IST).
- Favorites System: Users can mark parameters as favorites, and mini-graphs are shown in a dashboard preview for quick access.
- Settings Persistence: The platform will automatically remember user settings (favorites, selected parameter, and graph view) by storing in Lovableâ€™s internal database.

## Style Guidelines:

- Primary color: Deep navy blue (#0A2540) for titles and charts, representing stability and precision.
- Background color: Off-white (#F5F5F5) to provide a clean and modern feel.
- Accent color: Golden yellow (#FFC107) for highlights and icons, symbolizing energy and insight.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, objective look.
- Use a lightning bolt icon to represent electrical data, supplemented by parameter-specific icons. All should adhere to golden yellow accents.
- Employ a responsive layout with rounded cards and soft drop shadows for an enhanced interactive experience.
- Use subtle transitions and hover animations to provide feedback and improve usability.
